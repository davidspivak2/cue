# Subtitle fonts

Font files (`.ttf`, `.otf`) in this directory are registered with Qt at runtime so subtitle preview and overlay use the curated fonts (Heebo, Assistant, Rubik, IBM Plex Sans Hebrew, Noto Sans Hebrew, Alef, Arimo, Secular One, Suez One, Frank Ruhl Libre). Sourced from [Google Fonts](https://fonts.google.com/) (OFL/Apache-2.0).
